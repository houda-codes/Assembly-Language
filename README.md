# 🎵 Music Player using 8254 Timer (Assembly Language)

## 📌 Description

This project implements a simple music player using 8088 assembly language and the 8254 programmable interval timer. It generates sound signals to play the melody "Ode to Joy" through a speaker.

## ⚙️ Features

* Generates square wave signals using 8254 timer (Mode 3)
* Plays musical notes with different frequencies and durations
* Outputs notes visually on the terminal
* Uses frequency-to-note mapping technique
* Controls rhythm using delay procedures

## 🛠️ Technologies

* 8088 Assembly Language
* 8254 Programmable Interval Timer
* Microcomputer Interface

## 🔌 Hardware

* 8088 Microprocessor
* 8254 Timer
* Speaker
* Clock signal input

## ⚙️ How it Works

* Timer generates square wave signals
* Frequency determines pitch of the note
* Delay controls duration (beat)
* Notes are played sequentially to form a song

## 🚀 How to Run

1. Assemble the code using MASM/TASM
2. Load into emulator or hardware setup
3. Run the program
4. Listen to the generated melody

* Sound generation using square waves
